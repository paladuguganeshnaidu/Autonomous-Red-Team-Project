# RAG Knowledge Data

Place curated knowledge files in this directory for local RAG ingestion.

Recommended content groups:
- CVE notes and exploit preconditions
- Vulnerability pattern playbooks
- Payload strategy notes (not raw dumps)
- Validated past findings and remediation context

Supported file types:
- .md
- .txt
- .json

This folder is ingested at startup when RAG_BOOTSTRAP_ON_START is enabled.
